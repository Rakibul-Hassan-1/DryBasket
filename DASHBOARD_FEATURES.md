# ✅ Admin Dashboard - সম্পূর্ণ ফিচার লিস্ট

## 🎯 কী তৈরি হয়েছে?

### **Admin Dashboard** (`/admin-dashboard`)

একটি সম্পূর্ণ পেশাদার ড্যাশবোর্ড যেখানে Admin শুধুমাত্র লগইন করে সবকিছু ম্যানেজ করতে পারবেন। Admin-দের কোনো টেকনিক্যাল জ্ঞানের প্রয়োজন নেই - সবকিছু সহজ এবং ইউজার-ফ্রেন্ডলি।

---

## 📋 ড্যাশবোর্ডের 4টি সেকশন:

### 1️⃣ **📊 Overview (স্ট্যাটিস্টিক্স)**

```
┌─────────────────────────────────────────┐
│  📦 Total Products   │  🛒 Total Orders  │
│  30 items            │  45 orders        │
├─────────────────────────────────────────┤
│  💰 Total Revenue    │  👥 Total Users   │
│  $3,250.00          │  120 users        │

- সব স্ট্যাটের উপর Emoji থাকায় সুন্দর দেখায়
- লাইভ ডেটা Firebase থেকে আসে

### 2️⃣ **📦 Products Management**

```

─────────────────────────────────────────────────

- "+ Add Product" বাটন দিয়ে নতুন প্রোডাক্ট যোগ করা যায়

## 🔧 কীভাবে সেটআপ করবেন?

### Step 1: Environment Variable সেট করুন

`.env.local` ফাইলে যোগ করুন:

```
NEXT_PUBLIC_ADMIN_EMAIL=আপনার_ইমেইল@gmail.com
```

### Step 2: সেই ইমেইল দিয়ে রেজিস্টার করুন

- `/login` পেজে যান
- "Register" ট্যাবে যান
- আপনার তথ্য ভরে রেজিস্টার করুন
- সেই ইমেইল অবশ্যই `NEXT_PUBLIC_ADMIN_EMAIL` এর সাথে ম্যাচ করতে হবে

### Step 3: লগইন করুন

- আপনার ইমেইল এবং পাসওয়ার্ড দিয়ে লগইন করুন

### Step 4: Dashboard এক্সেস করুন

- নেভিগেশন বারে "📊 Dashboard" বাটন পাবেন
- ক্লিক করুন এবং ড্যাশবোর্ড খুলবে

## 🎨 ড্যাশবোর্ডের ডিজাইন ফিচার:

✅ **Professional Layout**

- Header এ "Admin Dashboard" টাইটেল এবং ডেটা লোডিং স্ট্যাটাস
- "Back to Shop" বাটন দিয়ে সহজে শপে ফিরে যেতে পারবেন

✅ **Tab Navigation**

- 4টি ট্যাব সহজে চেঞ্জ করা যায়
- Current ট্যাব highlight থাকে

✅ **Responsive Design**

- ডেস্কটপ, ট্যাবলেট, মোবাইল সবতে কাজ করে
- টেবিল মোবাইলে scroll হয়

✅ **Color Coding**

- স্ট্যাটাস ব্যাজ ভিন্ন ভিন্ন রঙে
- সহজেই বুঝা যায় কোনটা কী

✅ **Real-time Data**

- Firebase থেকে লাইভ ডেটা
- ড্যাশবোর্ড লোড হওয়ার সময় সব ডেটা ফেচ হয়

## 📊 যে সব ডেটা দেখা যায়:

### Overview থেকে:

- সব Products এর সংখ্যা
- সব Orders এর সংখ্যা
- মোট Revenue (সব অর্ডারের টোটাল)
- সব Users এর সংখ্যা

### Products থেকে:

- প্রোডাক্ট নাম, ক্যাটেগরি, দাম, স্টক
- স্টক স্ট্যাটাস (In Stock/Low/Out)
- নতুন প্রোডাক্ট যোগ করার অপশন

### Orders থেকে:

- অর্ডার ID
- কাস্টমার ইমেইল
- শিপিং তথ্য (নাম, ফোন, ঠিকানা)
- পেমেন্ট মেথড
- অর্ডার করা আইটেম
- অর্ডারের স্ট্যাটাস
- টোটাল এমাউন্ট

### Users থেকে:

- ইউজারের নাম
- ইমেইল
- ফোন
- ঠিকানা
- রোল (Customer/Admin)
