## ✅ 3 ধাপে শুরু করুন

### ধাপ 1️⃣ : Firebase সেটআপ

- https://console.firebase.google.com এ যান
- নিজের প্রজেক্ট বেছে নিন/তৈরি করুন
- Web App যোগ করুন
- Credentials কপি করুন এবং `.env.local` এ পেস্ট করুন

### ধাপ 2️⃣ : Admin Account তৈরি করুন

```
URL: http://localhost:3000/login
ট্যাব: Register
ইমেইল: admin@drybasket.com
পাসওয়ার্ড: DryBasket@123456
```

### ধাপ 3️⃣ : Dashboard অ্যাক্সেস করুন

```
URL: http://localhost:3000/admin-dashboard
OR
নেভিগেশন থেকে "📊 Dashboard" ক্লিক করুন
```
