### 3 ধাপে প্রবেশ করুন:

**ধাপ 1: Firebase সেটআপ করুন**

- https://console.firebase.google.com এ যান
- আপনার প্রজেক্টের credentials পান
- `.env.local` এ পেস্ট করুন

**ধাপ 2: Admin Account তৈরি করুন**

- http://localhost:3000/login এ যান
- Register ট্যাবে ক্লিক করুন
- উপরের credentials ব্যবহার করে রেজিস্টার করুন

**ধাপ 3: Dashboard অ্যাক্সেস করুন**

- লগইন করলে Navigation এ "📊 Dashboard" বাটন পাবেন
- ক্লিক করে Dashboard খুলুন

## 🎯 Dashboard এ কী কী পাবেন?

### 📊 Overview Tab

- মোট পণ্য সংখ্যা
- মোট অর্ডার সংখ্যা
- মোট আয় (Revenue)
- মোট ব্যবহারকারী

### 📦 Products Tab

- সব পণ্যের তালিকা (টেবিল ফরম্যাট)
- স্টক স্ট্যাটাস (In Stock/Low Stock/Out)
- নতুন পণ্য যোগ করার সুবিধা

### 🛒 Orders Tab

- সব গ্রাহক অর্ডার
- গ্রাহক তথ্য
- অর্ডার বিস্তারিত
- পেমেন্ট তথ্য

### 👥 Users Tab

- সব নিবন্ধিত ব্যবহারকারী
- ব্যবহারকারীর সম্পূর্ণ প্রোফাইল
- রোল তথ্য (Customer/Admin)
