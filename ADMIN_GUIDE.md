# 🛡️ Admin Dashboard Guide

## Admin Dashboard Features

Admin-দের জন্য একটি সম্পূর্ণ পেশাদার ড্যাশবোর্ড তৈরি করা হয়েছে যেখানে তারা শুধুমাত্র লগইন করে সবকিছু ম্যানেজ করতে পারবেন।

### 📊 Overview Tab

- **Total Products**: সমস্ত পণ্যের সংখ্যা
- **Total Orders**: সমস্ত অর্ডারের সংখ্যা
- **Total Revenue**: মোট আয়
- **Total Users**: নিবন্ধিত ব্যবহারকারীর সংখ্যা

### 📦 Products Tab

- সমস্ত পণ্যের তালিকা
- পণ্যের নাম, বিভাগ, মূল্য, স্টক দেখা যায়
- স্টক স্ট্যাটাস সহজে চেনা যায়:
  - 🟢 **In Stock**: ১০+ ইউনিট
  - 🟡 **Low Stock**: ১-১০ ইউনিট
  - 🔴 **Out**: ০ ইউনিট
- "+ Add Product" বাটন দিয়ে নতুন পণ্য যোগ করা যায়

### 🛒 Orders Tab

- সমস্ত অর্ডারের সম্পূর্ণ তথ্য
- প্রতিটি অর্ডারে দেখা যায়:
  - অর্ডার আইডি
  - গ্রাহকের ইমেইল
  - মোট মূল্য
  - **অর্ডার স্ট্যাটাস**: Pending, Shipped, Delivered
  - গ্রাহক তথ্য (নাম, ফোন, ঠিকানা)
  - পেমেন্ট পদ্ধতি
  - অর্ডার করা আইটেমগুলি

### 👥 Users Tab

- সমস্ত নিবন্ধিত ব্যবহারকারীর তালিকা
- প্রতিটি ব্যবহারকারীর তথ্য:
  - সম্পূর্ণ নাম
  - ইমেইল
  - ফোন নম্বর
  - ঠিকানা
  - রোল (Customer/Admin)

## 🔐 কীভাবে Admin Dashbaord অ্যাক্সেস করবেন?

1. আপনার ইমেইল `.env.local` ফাইলে `NEXT_PUBLIC_ADMIN_EMAIL` হিসেবে সেট করুন
2. সেই ইমেইল দিয়ে লগইন করুন
3. নেভিগেশন বারে "📊 Dashboard" লিঙ্ক দেখা যাবে
4. ক্লিক করুন এবং পুরো ড্যাশবোর্ড অ্যাক্সেস করুন
